우선 순위가 있어서 위에서 아래도 적용이 된다.

### 스타일 우선 순위

----------------------

```
얼마나 중요한가
	사용자 스타일
	제작자 스타일
	브라우저 스타일
적용 범위
	!important
	인라인 스타일
	id 스타일
	class 스타일
	타입 스타일 
소스 작성 순서 
	나중에 작성한 스타일이 먼저 작성한 스타일을 덮어 쓴다.
```
